
# AI_Agent_Learning
我的 AI Agent 开发学习记录仓库，目标是掌握大模型 Agent、RAG、LangChain 等核心技术，最终实现就业。

## 📚 学习目标
- 掌握 LangChain/LangGraph 开发工业级 AI Agent
- 实现 RAG 知识库问答、自动化办公 Agent 等实战项目
- 掌握 FastAPI + Docker 部署 Agent 服务

## 🛠 技术栈
- 核心框架：LangChain / LangGraph / FastAPI
- 向量数据库：Chroma / Pinecone
- 爬虫：BeautifulSoup4 / Playwright
- 部署：Docker / 云服务器
- 前端：Streamlit / Gradio
